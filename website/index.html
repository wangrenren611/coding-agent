架构设计

核心技术架构

采用分层架构设计，将复杂的 Agent 逻辑分解为独立的可插拔组件

应用层

CLI / Web UI / API

Agent 层

协调器 · ReAct 引擎 · 工具注册表

Provider 层

Provider 注册表 · HTTP 客户端 · 适配器

LLM 服务层

GLM · Kimi · MiniMax

Agent 协调器

管理任务生命周期，协调各组件工作，实现 ReAct 推理引擎

LLMCaller

封装 LLM 调用逻辑，处理流式响应，支持多 Provider 适配

ToolExecutor

工具执行调度，超时控制，结果处理，支持 16+ 内置工具

Session 会话管理

消息管理，上下文压缩，持久化触发，支持长对话处理

EventBus 事件总线

组件间解耦通信，发布订阅模式，支持多种事件类型

MemoryManager

持久化存储抽象，会话数据管理，支持文件存储后端
