# Claude Team 计划更新

**发布日期**：2026年1月28日  
**分类**：产品公告  
**阅读时间**：5 分钟

## 概述

Anthropic 今天让 Claude Team 计划对发展中的团队更易于访问。降低了价格并推出了高级座位的年度折扣。

## 定价更新

### 标准席位
- **新价格**：每月 $20（年度折扣）或每月 $25（按月计费）
- **使用量**：比 Pro 计划包含更多使用量

### 高级席位
- **新价格**：每月 $100（新增年度折扣）或每月 $125（按月计费）
- **使用量**：标准座位的 5 倍使用量

### 灵活容量
- 如果超级用户需要更多使用量，管理员可以为他们的团队购买额外容量

## Team 计划特性

### 团队协作
- 每个 Team 座位都有一个共享工作区
- 所有团队成员都可以在其中协作处理项目
- 通过连接器访问内部知识

### Claude Code
- 每个 Team 座位都包含 Claude Code
- 无论是用于发布功能的开发者还是用于自动化工作流的产品经理，都可使用

### 管理与计费
- 集中化的管理控制和计费使管理团队变得容易
- 数据保持私密——默认情况下不在你的内容上训练 Claude

## 使用场景

### 开发者
- 发布功能、编写代码、调试问题
- 使用连接器与代码库集成

### 产品经理
- 自动化工作流、创建文档、管理任务
- 使用 Claude Code 加速产品开发

### 市场营销
- 内容创作、市场分析
- 使用技能和工作空间管理营销活动

### 销售
- CRM 集成、客户跟进
- 使用 Claude 生成个性化销售内容

## 可用性

- 这些更改现已对所有用户开放
- 立即开始使用：[claude.com/team](http://claude.com/team)
- 更多信息：访问[帮助中心](https://support.claude.com/en/articles/9266767-what-is-the-team-plan)

## 总结

Claude Team 计划现在更加实惠，使发展中的团队能够更容易地协作并使用 Claude 强大的 AI 能力。该计划提供了共享工作区、Claude Code 集成以及简化的管理和计费功能。
