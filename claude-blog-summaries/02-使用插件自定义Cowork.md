# 使用插件自定义 Cowork

**发布日期**：2026年1月30日  
**分类**：产品公告  
**阅读时间**：5 分钟

## 概述

Anthropic 构建了 Cowork，将 Claude Code 的强大功能带给每个人。现在，新增了对插件的支持，插件让用户能够将任何技能、连接器、斜杠命令和子代理捆绑在一起，将 Claude 转变为专门针对特定角色、团队和公司的专家。

## 什么是插件

插件让用户可以：
- 告诉 Claude 喜欢如何完成工作
- 指定要使用哪些工具和数据
- 如何处理关键工作流
- 暴露哪些斜杠命令以便团队获得更好、更一致的结果

用户只需定义一次插件内容，Claude 就会在相关时从该上下文中提取信息。

## 插件在实践中的价值

### 销售示例
一个销售插件可以：
- 连接到 CRM 和知识库
- 教授 Claude 销售流程
- 提供从潜在客户研究到电话跟进的所有命令

### 跨功能专家
随着团队构建和共享插件，Claude 成为跨功能专家。共享的丰富上下文会被整合到每个相关交互中，因此领导者和管理员可以花费更少的时间来强制执行流程，而将更多时间用于改进流程。

## 开源插件市场

Anthropic 开源了 11 个由其自身团队构建和使用的插件：

1. **生产力** — 管理任务、日历、日常工作流程和个人上下文
2. **企业搜索** — 在公司的工具和文档中查找信息
3. **插件创建/自定义** — 从头开始创建和自定义新插件
4. **销售** — 研究潜在客户、准备交易并遵循销售流程
5. **财务** — 分析财务数据、构建模型并跟踪关键指标
6. **数据** — 查询、可视化和解释数据集
7. **法律** — 审查文档、标记风险并跟踪合规性
8. **营销** — 起草内容、规划活动并管理发布
9. **客户支持** — 分诊问题、起草回复并提出解决方案
10. **产品管理** — 撰写规范、确定路线图优先级并跟踪进度
11. **生物学研究** — 搜索文献、分析结果并规划实验

## 获取和使用

### 安装方式
- 直接从 Cowork 安装
- 浏览网站上的完整集合：[claude.com/plugins-for-cowork](https://claude.com/plugins-for/cowork)
- 上传自己的插件（可使用 Plugin Create 构建）

### 技术特性
- 插件的每个组件（技能、连接器、斜杠命令和子代理）都是基于文件的
- 插件易于构建、编辑和共享
- 可使用 Claude 自定义每个插件以使其完全符合用户和团队的需求

### 当前状态
- Cowork 中的插件支持目前作为研究预览版提供给所有付费 Claude 用户
- 插件目前保存到用户本地机器
- 对组织级共享和管理的更好支持（私人插件市场等）将在未来几周内推出

### 开发者资源
- 插件也在 GitHub 上可用：[github.com/anthropics/knowledge-work-plugins](https://github.com/anthropics/knowledge-work-plugins)

## 总结

插件为 Cowork 添加了一层强大的定制能力，使 Claude 能够：
1. 专门化针对特定工作职能
2. 与现有工具和数据集成
3. 标准化跨团队的输出和流程
4. 通过斜杠命令提供快速访问常用操作

无论是使用开源集合、自定义现有插件，还是构建全新的内容，插件都为组织使用 Claude 开启了新的可能性。
