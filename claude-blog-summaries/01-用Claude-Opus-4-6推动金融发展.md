# 用 Claude Opus 4.6 推动金融发展

**发布日期**：2026年2月5日  
**分类**：企业 AI  
**阅读时间**：5 分钟

## 概述

Claude Opus 4.6 标志着金融领域 AI 的重大进展。它可以帮助专业人士基于准确信息和清晰分析做出决策，并产出真正 polished 的成果。该模型在金融推理、多任务处理以及在更长的多步骤任务中保持专注方面，显著优于市场上的其他模型。

## 核心改进

### 研究能力
- 在 BrowseComp 和 DeepSearchQA 两个基准测试中表现提升，这两个测试评估模型从大型非结构化数据源中提取特定信息的能力
- 在 Finance Agent 基准测试中达到 60.7%（相比 Opus 4.5 提升 5.47%）
- 在 TaxEval 基准测试中达到 76.0%，成为该领域的 state-of-the-art

### 分析能力
Claude Opus 4.6 在金融分析领域处于领先地位，能够处理复杂的投资和财务分析任务。

### 创作能力
- 结构化输出（如电子表格和演示文稿）首次生成的准确率大幅提升
- 从 Claude Opus 4.5 到 Opus 4.6，输出质量有显著改善

## 产品更新

### Cowork
- 提供更 polished 的输出，如财务模型和演示文稿
- 支持同时启动多个分析，同时引导 Claude 的思维过程以符合用户标准

### Claude in Excel
- 更好地处理长期运行任务
- 新增支持：数据透视表编辑、图表修改、条件格式、排序和筛选、数据验证、金融级格式
- 新增性能改进：长对话自动压缩、拖放多文件支持

### Claude in PowerPoint
- 作为研究预览版（beta）推出
- 支持从客户端模板构建演示文稿
- 可对现有幻灯片进行有针对性的编辑
- 可从头生成出色的首次演示

## 内部验证

Anthropic 的内部 Real-World Finance 评估在约 50 个投资和财务分析用例中测量 Claude 的性能，涵盖电子表格、幻灯片演示和 Word 文档生成与审查。

Claude Opus 4.6 相比 Claude Sonnet 4.5（几个月前最先进的模型）提升了超过 23 个百分点。

## 使用说明

- Claude Opus 4.6、Cowork 和 Claude in Excel 可在所有付费 Claude 计划上使用
- Claude in PowerPoint 目前作为研究预览版供 Max、Team 和 Enterprise 用户使用
- 对于高风险工作，人工判断仍然至关重要，用户应继续审查 Claude 的输出以确保符合其规范
